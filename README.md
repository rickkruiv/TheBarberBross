## Visão do Sistema

Uma plataforma **SaaS multi-tenant** para:
- Redes de barbearias
- Controle financeiro avançado
- CRM de clientes
- Sistema de fidelidade
- Gestão de estoque
- BI com métricas estratégicas
- Aplicativo mobile para clientes
- Aplicativo mobile interno para barbeiros
- Módulo de portfólio para barbeiros e barbearias
- Módulo de comunidade de profissionais (Eventos, oportunidades e networking) 

## Complexidade Técnica
- Controle de permissões (RBAC)
- Arquitetura modular
- Dashboard com gráficos dinâmicos
- Sistema de pontuação/fidelidade
- Notificações push
- Relatórios exportáveis
- Logs de auditoria
- API documentada
- Testes automatizados
- Docker + CI/CD

## Escopo Completo

### 1. Gestão Multiunidade
- Cadastro de múltiplas unidades
- Gestão de funcionários por unidade
- Controle de metas por unidade
- Relatórios comparativos entre unidades

### 2. CRM de Clientes
- Histórico completo de serviços
- Ticket médio
- Frequência
- Classificação automática (cliente ouro, prata, bronze)
- Sistema de fidelidade com pontos
- Cupons de desconto

### 3. Gestão Financeira Avançada
- Controle de receitas
- Controle de despesas
- Fluxo de caixa
- Comissão automática por barbeiro
- Relatório DRE simplificado
- Exportação PDF/CSV

### 4. Gestão de Estoque
- Cadastro de produtos
- Controle de entrada e saída
- Alerta de estoque mínimo
- Relatório de consumo por serviço
- Cálculo de custo por atendimento

### 5. Inteligência de Negócio (BI)
Dashboard com:
- Receita por período
- Serviço mais lucrativo
- Barbeiro mais produtivo
- Horários mais movimentados
- Previsão simples de demanda (média móvel)

### 6. Aplicativo Mobile – Cliente
- Agendamento inteligente (mostra melhores horários)
- Sistema de pontos
- Carteira de cupons
- Avaliação de atendimento
- Histórico completo
- Notificações push

### 7. Aplicativo Mobile – Barbeiro
- Visualizar agenda do dia
- Confirmar atendimento
- Registrar pagamento
- Visualizar comissão
- Registrar venda de produto
